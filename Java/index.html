Java入门

教你选择Java

版本选择

当前造图端使用的版本是1.7.10，所以Java版本选择Java8，别称Jre8。

查看系统位数

如果你是Win7/8.1/10/11用户的话，直接进入计算机属性就能看到了。

鼠标右键计算机→点击属性，在系统类型里有显示。

如果实在找不到的话，建议使用官网的自动识别。

下载地址

可到官方网站下载，也可到下载页里去下载，搜索引擎里直接搜索也能搜到。

安装包版

1.获得Java安装包，双击打开

2.按照提示进行下一步直到按照结束即可


免安装版

1.获得Java免安装版包，解压到任意位置

2.进入我的世界启动器，手动添加Java路径
