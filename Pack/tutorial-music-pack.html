资源包入门

教你如何制作资源包，仅造图端版本(1.7.10)

音乐包 教程

入门级教程，内容一大部分采自WIKI，尽量让你搞懂。

我们先来看下，音乐包的结构

音乐包的结构
  • （资源包名称）
    • pack.mcmeta
    • pack.png(可选)
    • assets
      • minecraft
        • sounds.json
        • sounds
          • (声音).ogg

好的，我们正式开始制作吧！


1. 随便在一个地方，创建一个『文件夹』，『名称』自定

这个『文件夹』就是结构中的『（资源包名称）』，『名称』会显示在最上面

2. 进入『文件夹』，创建一个『文本文件』

这个『文本文件』，后缀名更改为『mcmeta』，这就是结构中的『pack.mcmeta』

pack.mcmeta内容为

{
"pack": {
"pack_format": 1,
"description": "音乐包"
}
}

其中『"pack_format"』如果1，无法识别，可以改成2.

其中『"description"』后面是，命名音乐包的『名称』的地方
会在游戏里的文件夹/压缩包名称下显示

(可选).找一张『尺寸』是『宽度和高度』相等的PNG『图片』

可以用PS或其他工具进行裁剪，然后将『文件名称』改为『pack』
这是音乐包的LOGO图片，这就是结构中的『pack.png』
注：这个LOGO图片不弄也行，不会影响识别。

3. 创建一个名为『assets』的文件夹

这是结构中的『assets』

4. 然后，进入『assets』文件夹，再创建一个名为『minecraft』的文件夹

这是结构中的『minecraft』

5. 然后，进入『minecraft』文件夹，再创建一个名为『sounds』的文件夹

这是结构中的『sounds』

6. 然后，进入『sounds』文件夹

选择，随便找一个音频，把它转成ogg格式，放进『sounds』文件夹内
音频名称自定，这是结构中的『(声音).ogg』

注：音频大小尽量不要超过8M，不然会卡住

7. 然后，返回『minecraft』文件夹下，创建一个sounds.json文件

这是结构中的『sounds.json』

sounds.json内容为

{
"wh": {
"category": "record",
"sounds": [
{
"name": "wh",
"stream": true
}
}
}

其中『"wh"』是音频文件名称

其中『"record"』是播放源，1.7.10默认record就行了

8. 现在，音乐包就制作完成了

可以进游戏内装上，使用下列指令验证音频是否能用

指令格式：/playsound 音频名称 玩家 x y z 音量/可听到距离
例如：/playsound wh @a ~ ~ ~ 10000

以上就是 音乐包 的教程了
如果你感兴趣的话，可用自己去尝试一下了！
其他具体的内容教程可进入 我的世界WIKI 学习
