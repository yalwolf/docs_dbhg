指令入门

向你介绍并使用，造图端（1.7.10）能用的指令！

scoreboard指令集

如果你知道scoreboard指令的话，自便。

命令的介绍

scoreboard指令集，在高级指令页面，已经给出了一些语法
分别是，计分板与队伍的创建以及修改，那么我在详细的将语法列一遍


命令的语法

计分板:/scoreboard objectives <list|add|remove|setdisplay>

计分板列表:
/scoreboard objectives list

计分板创建:/scoreboard objectives add <计分板名字> <计分板规格> <计分板显示的名字>

计分板移除:
/scoreboard remove <计分板名字>

计分板显示位置:/scoreboard setdisplay <位置> <计分板名字>

----------------------------------------------------------------------------------

计分板玩家数值:
/scoreboard players <list|add|remove|set>

玩家列表:
/scoreboard players list

增加玩家数值:
/scoreboard players add <玩家> <计分板名字> <数值>

移除玩家数值:
/scoreboard players remove <玩家> <计分板名字> <数值>

重置玩家数值:
/scoreboard players reset <玩家>

设置玩家数值:
/scoreboard players set <玩家> <计分板名字> <数值>

----------------------------------------------------------------------------------

队伍:
/scoreboard teams <list|add|remove|empty|join|leave|option>

队伍列表:
/scoreboard teams list

添加队伍:
/scoreboard teams add <队伍名> <显示的队伍名>

删除队伍:
/scoreboard teams remove <队伍名>

清空队伍:
/scoreboard teams empty <队伍名>

加入队伍:
/scoreboard teams join <队伍名> <玩家>

退出队伍:
/scoreboard teams leave <玩家>

队伍设置:
/scoreboard teams option <队伍名> <friendlyfire|color|seeFriendlyInvisibles> <数值>

名词解释

add 是 增加
set 是 设置/设定
list 是 列表
remove 是 删除/移除
reset 是 重置

empty 是 清空队伍内的全部群员
join 是 加入
leave 是 退出
option 是 队伍设置
friendlyfire 是 队友之间可否互相伤害
color 是 队伍颜色
seeFriendlyInvisibles 是 隐身队友是否可见
